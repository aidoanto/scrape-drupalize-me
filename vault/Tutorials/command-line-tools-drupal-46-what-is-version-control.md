# What Is Version Control?

## Content

Sprout Video

## Overview

In this tutorial we'll explain some of the basic tenets of version control systems. We'll define Git terminology like repositories, branches, checkouts, and commits, and provide you with a baseline set of concepts that we can build on throughout the course.

## Learning Objectives

- Understand the basic terminology of version control including repository, branch, checkout and commit.
- Understand how these concepts relate to Git.
- Be able to explain the difference between centralized and distributed version control systems.
